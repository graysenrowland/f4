# F₄ Site Agent Notes

This repository contains the static site for the Freedom Family Film Festival (F₄).

## Working rules

- Do not add or remove site sections, links, resources, dates, eligibility language, awards, requirements, or other content unless Gray and the assistant have explicitly agreed to the change in conversation.
- Once Gray and the assistant have agreed on a specific change, proceed actively with the edit instead of restating the plan or waiting for another confirmation.
- Prefer edits that preserve the existing structure and visual pattern unless a structure change has been approved.
- Keep copy concise and clear for students, families, and emerging filmmakers.
- Keep registration links centralized through `window.F4_CONFIG.registrationUrl` in `index.html`.
- Use local files in `/resources` when available rather than external asset links.
- Do not include verbose commit details in responses unless requested.

## Current site style

- Single-page static site for GitHub Pages.
- Main content uses unified panel/timeline-style sections.
- CSS lives in `assets/css/style.css`.
- Resource files live in `resources/`.
